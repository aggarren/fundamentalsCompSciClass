Homework0.cc is the C++ code for the program

Homework0.txt has the test cases for the above program

Any test cases are labeled test# corresponding with the above file's contents

To compile program:
   type in command prompt: g++ -c Homework0.cc
   then type g++ Homework0.o -o Homework0
   execute by typing ./Homework0

To execute test cases:
   type in command prompt: ./Homework0 < test#.txt

To store output from test cases after executing:
   type in command prompt: ./Homework0 < test#.txt > out#.txt
