Project3.cc - main program file
functions.cc - simulation preparation and population files and helper functions
Simulator.h - Simulator class declaration
Simulator.cc - functions to make data and run Simulator
Cust.h - Cust class declaration
Teller.h - Teller class declaration
Teller.cc - Teller class functions
CustData.h - CustData class declaration - CustData is a Queue of Cust objects
Queue.h - Queue class declaration - sub class of LinkedList class
LinkedList.h - LinkedList class declaration
Node.h - Node class declaration
test1-data, etc - test case files

To compile and link:
type in command line: make Project3

To run:
Debug mode: ./Project3 d test1.data
Simulation mode: ./Project3 s time tellers iterations

Outputs:
Output Stream
Customers.results - in file folder once run
Teller-i.results - in file folder once run
